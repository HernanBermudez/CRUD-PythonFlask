# Proyecto-Equipos-Flask
Este proyecto consiste en una REST API usando Flask. Es un CRUD de una base de datos MySQL, la cual contiene los datos de los 10 mejores equipos de fútbol según la UEFA. El usuario podrá seguir agregando elementos al ranking si lo desea para probar su funcionalidad. El deployment del proyecto se realizó en Heroku.

Mira el proyecto aquí: https://football-teams-pyflask.herokuapp.com/
